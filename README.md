# 🚦 Traffic Demand Prediction

## 📌 Problem Statement
Predict traffic demand at various urban locations using 
location, road, and weather features.

## 📊 Dataset
- Train: 77,299 rows × 11 columns
- Test: 41,778 rows × 10 columns
- Source: HackerEarth Machine Learning Challenge

## 🔧 Features Used
- Geohash (location encoding)
- Day & Timestamp
- Road Type & Number of Lanes
- Large Vehicles allowed or not
- Landmarks nearby
- Temperature & Weather

## 🤖 Approach
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training & Evaluation
- Metric: R² Score
